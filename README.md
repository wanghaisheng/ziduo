# handle-mini

## 依赖安装

```bash
pnpm install
```

## 本地开发

```bash
pnpm start
```

## 代码检查

```bash
pnpm run lint:script
pnpm run lint:style
```

## 代码检查及修复

```bash
pnpm run lint:script --fix
pnpm run lint:style --fix
```

## 类型检查

```bash
pnpm run type
```

## 测试

```bash
pnpm test
```

## 生产构建

```bash
pnpm run build
```
